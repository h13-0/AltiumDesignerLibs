# AltiumDesignerLibs
自用的个人绘制的AD的原理图、封装图库。
使用时直接在本页搜索关键词即可。
原理图库位于 `SchLib` 目录下，封装库位于 `PcbLib` 下。
部分原件未施工完毕，仅供自用。
